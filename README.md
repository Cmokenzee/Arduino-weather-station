Arduino Weather Station
A compact weather station built with Arduino Nano that measures temperature, humidity and atmospheric pressure, displays readings on an OLED screen and logs data to an SD card.

About
This project is a fully functional weather station that reads environmental data from two sensors (DHT22 for temperature/humidity, BMP280 for pressure/altitude), displays the current readings on a 0.96" OLED display, and logs historical data to a micro SD card in CSV format for later analysis.
The station cycles through different display screens — temperature, humidity, pressure, and a combined overview — updating every few seconds. Logged data includes timestamps and can be opened in Excel or any spreadsheet tool.
